## Como usar o agente

```
/jtp-doc lavagem          → gera POP + fluxograma da tela de lavagem
/jtp-doc abastecimento    → gera POP + fluxograma do abastecimento
/jtp-doc pcm-os           → gera POP + fluxograma da OS (desktop)
/jtp-doc rede-apoio       → gera/atualiza rede de apoio e protocolo de acionamento
/jtp-doc update           → verifica tudo que mudou desde a última geração e atualiza
/jtp-doc all              → gera documentação completa (consome mais tokens — usar com cuidado)
```
