# Código completo — troca prévia de carro na soltura 2.10

Para aplicar a funcionalidade completa no GitHub, substitua integralmente estes arquivos do repositório pelos arquivos desta branch/commit:

1. `app-mobile.html`
2. `index.html`

## O que foi incluído

- No mobile (`app-mobile.html`):
  - badge `TROCA PRÉVIA` no card quando a escala já foi alterada previamente;
  - botão `Trocar Carro Previamente` para registros pendentes;
  - modal/listagem de veículos disponíveis para escolher o carro reserva;
  - atualização de `prefixo`, `veiculo_id` e observação mantendo a saída como pendente.

- No desktop (`index.html`):
  - botão `🔄 Trocar` em registros pendentes da soltura 2.10;
  - função de troca prévia com validação de carros disponíveis;
  - indicador `🔄 Prévia` quando a escala possui troca antecipada registrada.

## Como substituir no GitHub

1. Abra o arquivo `app-mobile.html` no GitHub.
2. Clique em editar.
3. Apague todo o conteúdo atual.
4. Cole o conteúdo completo do `app-mobile.html` desta branch/commit.
5. Salve o arquivo.
6. Repita o mesmo processo para `index.html`.

> Importante: substitua o arquivo inteiro, não apenas trechos, porque a funcionalidade depende de funções JavaScript e marcações HTML que trabalham juntas.
